## Usage

```
<div class="card">
  <header>
    <h2>Card Title</h2>
    <p>Card Description</p>
  </header>
  <section>
    <p>Card Content</p>
  </section>
  <footer>
    <p>Card Footer</p>
  </footer>
</div>
```

The component has the following HTML structure:

`<div class="card">`

The card container.

`<header>`

The header of the card.

`<h2>`

The title of the card.

`<p>` Optional

The description of the card.

`<section>`

The content of the card.

`<footer>` Optional

The footer of the card.
