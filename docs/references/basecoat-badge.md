# Badge | Basecoat

Displays a badge or a component that looks like a badge.

Badge

## [Usage](#usage)

Use one of the badge classes: `badge`, `badge-primary`, `badge-secondary`, `badge-destructive` or `badge-outline`.

```
<span class="badge">Badge</span>
```
